== Description ==

Advanced Image Grabber Import all images into your media library from external webpage urls.

### The features of the plugin

Advanced Image Grabber can import all images into your media library from external webpage urls.
 You can select the images to upload and import selected images automatically by one click.
 This plugin is extremely useful for the people who doesn't want to spend hours to search image in web and manually upload to wordpress media library.
 You can use Advanced Image Grabber at frontend inside post/page or theme with shortcode [aig_frontend] or php functions too. 

* Easy to use
* Import images from up to 5 urls at single click
* See live import status , success and failure
* Browse all imported image on click from same page
* Keeps orignal image name when importing
* Fully responsive and one click image selection
* Have shortcode to use the grabber at frontend
* Easy to use for developer
* Simple admin page at tools->Image Grabber

== Installation ==

### First download the ZIP file ,

You can either install it automatically from the WordPress admin, or do it manually.

1. Unzip the archive and put the 'advanced-image-grabber' folder into your plugins folder (/wp-content/plugins/)
2. Activate the plugin through the 'Plugins' menu in WordPress
3. You should now see Image Grabber menu under tools at your dashboard
